# Spinbazar - Backend

## Leírás

A **SpinBazar** egy iskolai vizsgaremek projekt, a backend egy express szerver, az adatbázis pedig mongoDB. [Frontend](https://github.com/PillerDaniel/SpinBazarFrontend)

## Használt package-ek

- mongoose
- card-validator
- jsonwebtoken
- nodemailer
- bcryptjs
- express-validator
